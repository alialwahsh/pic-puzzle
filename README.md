# PicPuzzle

PicPuzzle is a simple image sliding puzzle game built using HTML, CSS, and JavaScript. It runs entirely in the browser and does not require any backend.

## Features

- Upload any image to turn it into a puzzle  
- Select puzzle size (3x3, 4x4, or 5x5)  
- Tiles are shuffled in a solvable way  
- Move tiles using mouse clicks or arrow keys  
- Tracks time and number of moves  

## How to Run

1. Download or clone the project  
2. Open `index.html` in any web browser  
3. Upload an image and start playing
